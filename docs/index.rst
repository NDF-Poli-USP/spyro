
Spyro
=================================

Welcome to spyro's documentation!
spyro is a Python library for modeling acoustic waves. The main functionality is a set of forward and adjoint wave propagators for solving the acoustic wave equation in the time domain. 

Getting started
---------------

You can get instructions on how to download and install spyro
:doc:`here <installation>`

To learn how to effectively use it for wave simulation and full waveform inversion, read our :doc:`notebooks <notebooks>`
